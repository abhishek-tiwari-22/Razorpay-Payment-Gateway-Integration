# Razorpay-Payment-Gateway-Integration
Payment Gateway Integration made using Django as backend and HTML as frontend. Used SQLite.
After successfull payment Email will be recieved to user's mail id which is accessed using SMTP concept.

...

User will provide details like name, email and amount to be paid after providing the details,
user will submit and then a payment button will be generated click on that and select payment option and pay.
Select payment status and you will see the page with message of successfull payment.
After that check your mail, you will recieve a mail with notification of user's payment status.
